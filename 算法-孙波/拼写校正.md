## 查询矫正
### 词独立法
假设一：对需要纠错的词存在一系列“正确单词形式”
假设二：

单词间距离计算：
- 编辑距离
	- Levenshtein距离：基本操作是插入、删除和替换
	- Damerau-L：包括交换
	- 比较`s[i]`和`t[i]`的关系
	- 定义四象限：右下角始终为编辑距离
```c++
if s[i] = t[j]
	then m[i,j] = min{m[i-1][j] + 1, m[i][j-1] + 1, m[i-1][j-1]}
else 
	m[i,j] = min{m[i-1][j] + 1, m[i][j-1] + 1, m[i-1][j-1] + 1}
```
